## Installation

First, download the repository:

```bash
  git clone https://github.com/thalesnishida/api-movies
```

Open the folder, and type the command for install the dependencies:
```bash
  cd api-movies
  npm install 
```

Now type the command:

```bash
  npm run dev 
```
